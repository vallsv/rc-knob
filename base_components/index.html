Base components

The display part of the knob is splitted into the Pointer, Scale, Arc, Value and few others.

Pointer​

This component displays the location of the knob.

Three shapes are available circle, rect, triangle.

Arc​

This component displays 2 angular zones. The foreground one is from the min to the knob location. The background one is from the knob location to max.

Scale​

This component draw graduation.

Value​

This component draw the value of the knob.

Label​

This component draw a label at a fixed location using polar coordinates.

Range​

This component draw a range between two values using polar coordinates.
