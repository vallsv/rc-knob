The `Value` component draw the value of the knob.

```jsx live
<Knob size={100} angleOffset={220} angleRange={280} min={0} max={100}>
    <Value marginBottom={10} className="my-value-style" />
</Knob>
```

## Properties

<table>
    <tbody>
        <tr>
            <th>Prop</th>
            <th>Description</th>
        </tr>
        <tr>
            <td>`decimalPlace`</td>
            <td>
                Number of decimal places the value should rendered with. `0` by
                default.
            </td>
        </tr>
        <tr>
            <td>`className`</td>
            <td>`className` that is passed to the SVG `text` component.</td>
        </tr>
        <tr>
            <td>`marginBottom`</td>
            <td>Useful to adjust the horizontal position of the text inside</td>
        </tr>
    </tbody>
