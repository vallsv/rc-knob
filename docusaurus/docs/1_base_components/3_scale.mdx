description: Scale component
---

The `Scale` component draw graduation.

```jsx live
<Knob
    size={200}
    angleOffset={220}
    angleRange={280}
    min={0}
    max={100}
    steps={10} // amount of intervals
>
    <Scale
        tickWidth={5}
        tickHeight={10}
        radius={45}
        color="#FC5A96"
        activeColor="#180094"
    />
</Knob>
```
