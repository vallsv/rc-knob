title: Label
description: Label component
---

The `Label` component draw a label at a fixed location
using polar coordinates.

```jsx live
<Knob size={100} angleOffset={220} angleRange={280} min={0} max={100}>
    <Label
        percentage={0.0}
        radius={40}
        label="BEGIN"
        style={{ fontSize: '50%' }}
    />
    <Label
        percentage={0.5}
        radius={40}
        label="MIDDLE"
        style={{ fontSize: '50%' }}
    />
    <Label
        percentage={1.0}
        radius={40}
        label="END"
        style={{ fontSize: '50%' }}
    />
</Knob>
```

## Properties

<table>
    <tr>
        <th>Prop</th>
        <th>Description</th>
    </tr>
    <tr>
        <td>`className`</td>
        <td>Will be added to the SVG `text` element.</td>
    </tr>
    <tr>
        <td>`label`</td>
        <td>Label to display.</td>
    </tr>
    <tr>
        <td>`percentage`</td>
        <td>Location of the label in percentage of the value range</td>
    </tr>
    <tr>
        <td>`radius`</td>
        <td>Radius of the center of the label</td>
    </tr>
    <tr>
        <td>`style`</td>
        <td>Will be added to the SVG `text` element.</td>
    </tr>
    <tr>
        <td>`userSelect`</td>
        <td>Added to the style. Default is `none` to avoid text selection.</td>
    </tr>
