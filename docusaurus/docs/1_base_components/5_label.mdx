---
title: Label
description: Label component
---

The `Label` component draw a label at a fixed location
using polar coordinates.

```jsx live
<Knob size={100} angleOffset={220} angleRange={280} min={0} max={100}>
    <Label
        percentage={0.0}
        radius={40}
        label="BEGIN"
        style={{ fontSize: '50%' }}
    />
    <Label
        percentage={0.5}
        radius={40}
        label="MIDDLE"
        style={{ fontSize: '50%' }}
    />
    <Label
        percentage={1.0}
        radius={40}
        label="END"
        style={{ fontSize: '50%' }}
    />
</Knob>
```
